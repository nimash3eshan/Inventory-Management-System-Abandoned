Installation Process

Pre-Requirements

  • Create a new database , Make sure database is completely blank.
  • The maximum execution time of your server can't be less than 5 minutes. You may update php.ini file for this
  • To get Installation service from our Expert Support Team contact us at flexibledesign01@gmail.com

You have to follow this article carefully

1st : Install a software according your Operating System

Installation of WAMP for Windows

  • Downlaod and install WAMP software in your computer
  • Go to c: drive > WAMP folder> www folder > paste FlexiblePos folder(This you will get in our package).
  • Run WAMP software

Installation of XAMPP for Windows

  • Download and install XAMPP software in your computer
  • Go to c: drive >XAMPP folder> htdocs folder > paste FlexiblePos folder(This you will get in our package).
  • Run XAMPP software and start Apache and MySQL from XAMPP control panel

Installation of MAMP for MAC

  • Go to the link and download MAMP. Once the download is finished, open the downloaded file. Drag and drop the image file inside it to your Applications folder.
  • Paste FlexiblePos folder (This you will get in our package) your MAMP document root folder
  • Run MAMP

Installation of LAMP for Linux

    https://sourceforge.net/directory/?q=lamp%20server
  • Go to the link and download LAMP. Once the download is finished install it.
  • Paste FlexiblePos folder (This you will get in our package) your LAMP document root folder
  • Run LAMP
